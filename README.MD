# Improv Prompt Game

A tiny Webserver + bare html/css/js Website that lets you get random Improv Prompts anywhere, anytime!
<br>
NO vibe coding was used in the process of making this, though I acknowledge that an LLM would have probably put out something more professional looking. Well, you get what you pay for.
<br>
The Prompts were taken from Dropout's "Make Some Noise" game show by crawling [the fandom wiki](https://dropout.fandom.com/wiki/Make_Some_Noise).
<br>
If you want to run this yourself for some reason, all you need to do is clone the repository to a webserver that has docker installed, and run ``docker compose build && docker compose up -d``.
<br>
Or you can find my current deployment [here](http://improvgame.online/)!
